Login & Sign Up Form

A modern and responsive Login & Sign Up Form built using HTML, CSS, and JavaScript. This project features a clean user interface, smooth animations, and a mobile-friendly responsive design.

🌐 Live Demo

https://theprasun-codes.github.io/Login-and-sign-up-form/

✨ Features

- Modern UI Design
- Login & Registration Forms
- Smooth Transition Animations
- Fully Responsive Layout
- Mobile Friendly
- Lightweight and Fast
- Beginner Friendly Code
- Easy to Customize

🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/ThePrasun-codes/Login-and-sign-up-form.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
